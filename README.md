# Energy Forecasting and Scenario Modeling

## Project Overview
This project analyzes U.S. energy consumption data and builds forecasting models using ARIMA and Prophet. It also includes scenario-based modeling to simulate different future conditions.

## Objective
To predict future energy consumption and evaluate multiple demand scenarios using time-series techniques.

## Dataset
U.S. Energy Information Administration (EIA) dataset containing monthly energy consumption data.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- ARIMA (statsmodels)
- Prophet

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. ARIMA Modeling
4. Prophet Modeling
5. Model Evaluation
6. Multi-Year Forecasting
7. Scenario-Based Modeling

## Results
- Prophet performed better in capturing trend and seasonality.
- Forecast shows stable long-term energy consumption pattern.
- Scenario analysis highlights demand variations under different conditions.
